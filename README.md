## Proyecto

El proyecto está configurado usando [Maven](https://maven.apache.org/),
y puede ser compilado, empaquetado y probado fácilmente usando los siguientes comandos:

### Compilar

```
mvn compile
```

### Empaquetar

```
mvn package
```

### Pruebas

```
mvn test
```
